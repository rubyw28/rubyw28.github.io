DORA Laser Tracking Control System

As a Software Engineering and Research Intern at NASA JPL, I developed a Python-based full-stack control system for the Deployable Optical Receiver Aperture (DORA), integrating MQTT communication and real-time telemetry to control precision laser tracking. I built RESTful APIs and an interactive React dashboard for live status monitoring, configuration editing, and command input. I also implemented Bit Error Rate (BER) analysis using NumPy and Pandas, and wrote integration tests to ensure reliable communication with embedded systems.
